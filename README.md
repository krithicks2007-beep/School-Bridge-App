# School Parent App

React Native frontend for a school parent communication app.

## Current Screen

- Login screen only
- Blue and white visual theme
- No backend connection
- Replace `assets/logo.png` and `assets/background.png` with the final images

## Run

```bash
npm install
npm start
```

## Run on this laptop

```bash
npm run laptop
```

Then open:

```text
http://localhost:19006
```
